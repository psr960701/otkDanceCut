# 将默认终端设置为Git Bash

## 步骤说明

1. **打开命令面板**：
   - 按下快捷键 `Ctrl + Shift + P`
   - 或者点击顶部菜单 "View" > "Command Palette"

2. **搜索终端配置**：
   - 在命令面板中输入 `Terminal: Select Default Profile`
   - 选择该选项

3. **选择Git Bash**：
   - 在弹出的选项中，选择 `Git Bash`
   - 如果没有看到Git Bash选项，可能需要先安装Git

4. **验证设置**：
   - 关闭当前所有终端窗口
   - 重新打开终端（`Ctrl + ~` 或点击菜单栏 "Terminal" > "New Terminal"）
   - 新打开的终端应该会默认使用Git Bash

## 注意事项

- 确保已经安装了Git（https://git-scm.com/）
- 如果Git Bash没有出现在选项中，可能需要重启IDE
- 修改设置后，需要关闭并重新打开终端才能生效
